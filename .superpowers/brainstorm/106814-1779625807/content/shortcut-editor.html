交互流程
打开
托盘右键 → "快捷键设置" → 弹出独立窗口

修改快捷键
点击 ✏ → 该行进入录制模式（闪烁"按下组合键..."）→ 按下新组合 → 自动保存

冲突检测
如果新组合已被占用 → 弹出确认提示 → 用户确认后替换，旧功能快捷键清空

取消录制
录制模式下点 ✕ / 按 Esc → 取消，保持原快捷键

恢复默认
点击底部"恢复默认" → 所有快捷键回到初始值
