生活
交互行为
打开 — 点击卡片上 "📎 N 个附件"
打开文件 — 点击文件名 → 系统默认程序
删除 — 点击 ✕ → 仅取消关联，不删原文件
添加 — "+ 添加附件" 按钮 → 系统文件选择器
关闭 — ✕ 按钮 / 点击 popover 外 / Esc
拖拽 — 整个 popover 区域静默接受拖入（无视觉提示）
设计决策
紧凑 — popover 宽度 240px，无标题浪费空间
无拖拽区 UI — 不用虚线框，不用"拖到此处"提示
静默拖入 — drop 事件直接处理，不预先告知
干净 — 文件行无背景框，hover 时才显示底色
