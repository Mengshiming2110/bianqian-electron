交互细节
打开
点击卡片上的 "📎 N 个附件" → popover 从附件区域下方弹出
查看与打开
每行显示文件图标 + 文件名（截断超长部分），点击文件名 → 调用系统默认程序打开
拖拽添加
从 Windows 资源管理器拖文件到虚线区域 → 自动追加到附件列表
按钮添加
点击 "+ 添加附件" → 调起系统文件选择器（多选）
删除
点击每行右侧 ✕ → 从附件列表移除（仅移除引用，不删除原文件）
关闭
点击 ✕ 关闭按钮 / 点击 popover 外部区域 / 按 Esc

✗ 文件缩略图/预览
✗ 面板内拖拽排序
